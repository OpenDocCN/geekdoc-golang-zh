# Go (Golang)中指针的默认零值

> 原文:[https://golangbyexample . com/default-零值-指针-golang/](https://golangbyexample.com/default-zero-value-pointer-golang/)

# **概述**

指针的默认零值是零。让我们为它看一个程序

# **程序**

```
package main

import "fmt"

func main() {
    var a *int
    fmt.Print("Default Zero Value of a pointer: ")
    fmt.Println(a)
}
```

**输出:**

```
Default value of pointer: 
```
