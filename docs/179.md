# 围棋中的指针算法

> 原文:[https://golangbyexample.com/pointer-arithmetic-golang/](https://golangbyexample.com/pointer-arithmetic-golang/)

# **概述**

指针运算在 golang 中是不可能的，不像 C 语言。它会引发编译错误。

# **程序**

```go
package main
func main() {
    a := 1
    b := &a
    b = b + 1
}
```

**输出**

上述程序引发编译错误

```go
invalid operation: b + 1 (mismatched types *int and int)
```
