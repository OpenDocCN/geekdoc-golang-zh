# **概述**

GO 的**数学**包提供了一个 **Cbrt** 方法，可以用来得到那个数的立方根。

下面是函数的签名。它接受一个浮点数的输入，也返回一个浮点数。

```go
func Cbrt(x float64) float64
```

Cbrt 函数的一些特殊情况是

*   Cbrt( 0) = 0
*   Cbrt( Inf) = Inf
*   Cbrt(NaN) = NaN

# **代号:**

```go
package main

import (
    "fmt"
    "math"
)

func main() {
    res := math.Cbrt(8)
    fmt.Println(res)

    res = math.Cbrt(27)
    fmt.Println(res)

    res = math.Cbrt(30.33)
    fmt.Println(res)
}
```

**输出:**

```go
2
3
3.118584170228812
```
