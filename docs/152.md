```

```

在主功能中，我们正在等待**结果**通道来收集结果

```

```

该行将一直等到某个值被推送到**结果**通道。如上图所示， **sumValue** 将被 **sum** 功能推送到**结果**通道。

程序给出同样的输出，证明**主**功能等待 **sumValue** 推送到**结果**通道。一旦它从**结果**通道接收到值，它就打印它。
