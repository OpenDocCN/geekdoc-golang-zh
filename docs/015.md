# Go (Golang)中的内/外范围不变

> 原文：<https://golangbyexample.com/constant-scope-inner-outer-go/>

# **概述**

在内部范围内声明的常数与在外部范围内声明的常数同名，这将隐藏外部范围内的常数。

# **例**

```go
package main
import "fmt"
const a = 123
func main() {
    const a = 456
    fmt.Println(a)
}
```

**输出**

```go
456
```

上面的程序打印 456 作为输出，因为声明的内部常数有那个值
