# 圆周率

> 原文：<https://golangbyexample.com/pi-value-golang/>

# **概述**

**数学**包的 GO 提供了一个 `Pi` 常量。

数学包的 constant.go 文件中是这样定义的。这是一个浮点值

```go
Pi  = 3.14159265358979323846264338327950288419716939937510582097494459
```

# **代号:**

```go
package main

import (
	"fmt"
	"math"
)

func main() {
	pi := math.Pi
	fmt.Println(pi)
} 
```

**输出:**

```go
3.141592653589793
```
