# 在 Go (Golang)中的数组中查找和删除

> 原文:[https://golangbyexample.com/find-delete-array-golang](https://golangbyexample.com/find-delete-array-golang)

可能有两种情况:

# **修改原阵**

*   继续复制到原始数组，跳过需要删除的项目

*   结尾是 Reslice

```go
package main

import "fmt"

func main() {
    s := [4]int{1, 2, 3, 1}
    after := findAndDelete(s, 1)
    fmt.Println(after)
}

func findAndDelete(s [4]int, item int) []int {
    index := 0
    for _, i := range s {
        if i != item {
            s[index] = i
            index++
        }
    }
    return s[:index]
}
```

**输出:**

```go
[2,3]
```

# **不修改原阵**

创建一个新数组，并不断插入其中。

```go
package main

import "fmt"

func main() {
    before := [4]int{1, 2, 3, 1}
    after := findAndDelete(before, 1)
    fmt.Println(after)
}

func findAndDelete(s [4]int, itemToDelete int) []int {
    var new [4]int
    index := 0
    for _, i := range s {
        if i != itemToDelete {
            new[index] = i
            index++
        }
    }
    return new[:index]
}
```

**输出**

```go
[2,3]
```
