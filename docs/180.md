# Go (Golang)中指针的默认零值

> 原文：<https://golangbyexample.com/default-zero-value-pointer-golang/>

# **概述**

指针的默认零值是零。让我们为它看一个程序

# **程序**

```go
package main

import "fmt"

func main() {
    var a *int
    fmt.Print("Default Zero Value of a pointer: ")
    fmt.Println(a)
}
```

**输出:**

```go
Default value of pointer: 
```
