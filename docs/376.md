# 删除 Go (Golang)中的文件

> 原文：<https://golangbyexample.com/delete-file-go>

**os。Remove()** 功能可以用来删除 Golang 中的一个文件。下面是这个函数的签名

```go
func Remove(name string) error
```

**代码:**

```go
package main

import (
    "log"
    "os"
)

func main() {
    err := os.Remove("sample.txt")
    if err != nil {
        log.Fatal(err)
    }
}
```

**输出:**

```go
Deletes sample.txt file from the current working directory
```
