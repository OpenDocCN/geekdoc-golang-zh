# Golang中布尔值或打印布尔值的格式说明符

> 原文：<https://golangbyexample.com//>format-指定-for-bool-golang

不同的格式说明符可以用来打印布尔值或字符串。

*   **%t** 可用于以布尔形式打印
*   **%v** 将打印默认字符串。“真”代表真，“假”代表假

**代码:**

```go
package main

import "fmt"

func main() {
	t := true
	f := false

	fmt.Printf("%t %t\n", t, f)
	fmt.Printf("%v %v\n", t, f)
}
```

**输出**

```go
true false
true false
```
